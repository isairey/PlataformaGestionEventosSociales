# 🌌 Acerca del proyecto

**Event Planner React App** es una plataforma frontend orientada a la organización de eventos sociales como bodas, fiestas privadas, cumpleaños, reuniones empresariales y celebraciones especiales.

El sistema fue diseñado para:

- 🎉 Gestionar eventos sociales
- 👥 Administrar invitados
- 📅 Organizar reservaciones
- 📦 Gestionar servicios
- 💳 Controlar pagos
- 🔐 Administrar usuarios
- 📊 Supervisar actividades
- 🌐 Facilitar la planificación de eventos

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/PlataformaGestionEventosSociales.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd PlataformaGestionEventosSociales
```

---

## 3️⃣ Instalar dependencias

```bash
yarn install
```

o

```bash
npm install
```

---

## 4️⃣ Ejecutar entorno de desarrollo

```bash
yarn start
```

o

```bash
npm start
```

---

## 5️⃣ Abrir aplicación

```bash
http://localhost:3000
```
